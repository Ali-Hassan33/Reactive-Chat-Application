## About
Build a reactive chat application using Project Reactor, Spring Boot, and Vaadin, enabling real-time messaging among users.

## Development Environment
- Java 21
- Spring Boot 3.2.4
- Vaadin 24.3.8
- Project Reactor 3.6.4
- Maven
- PostgreSQL
- Intellij Idea
